### Các lỗi thường gặp
**Page not found**
=> Kiểm tra lại URL (Mặc định: http://localhost/webshopphp/public/index.php)

**Not found coulunm**
=> Không tìm thấy bảng trong database. Kiểm tra lại đã import đúng database chưa, chú ý tên database (web_shop_php) 

**Page Crash** 
=> Bật Xampp lên
